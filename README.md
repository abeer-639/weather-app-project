# Weather App 🌤️

A React weather application that shows real-time weather data.

## Live Demo
[View Live]    

## Features
- Search weather by city
- Dynamic background based on weather
- Shows humidity, wind speed, and feels like temperature

## Tech Stack
- React
- OpenWeather API
- CSS3

## Getting Started
1. Clone the repo
2. Run `npm install`
3. Add your API key in `.env` as `VITE_API_KEY`
4. Run `npm run dev`
